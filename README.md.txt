# Fun Learning for Kids

An educational game website for toddlers and preschool-aged children (ages 3 to 5) to learn colors, shapes, numbers, letters, logic, and motor skills through interactive games.

## Features
- 35 interactive games with drag-and-drop, click-to-match, and more.
- Responsive design using Bootstrap.
- Offline support with Service Workers.
- Background music with a toggle option.
- Simple scoring system with local storage for progress.
